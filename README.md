## Installation

To manage symlinks I use [toml-bombadil](https://github.com/oknozor/toml-bombadil), take a look :).
 
 ```bash
git clone https://github.com/oknozor/dotfiles.git
bombadil install dotfiles
bombadil link -p ayu-dark laptop|desktop niri
```
